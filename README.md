CUDA/C++ implementation of "The material point method for Snow Simulation" - https://www.math.ucla.edu/~jteran/papers/SSCTS13.pdf

Dependencies for the project include:
  - GLEW
  - GLFW
  - CUDA 7
  - GLM
  - DevIL
